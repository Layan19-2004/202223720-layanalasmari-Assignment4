# Layan Alasmari Portfolio вЂ“ Assignment 4

## Project Description

This project is a personal portfolio web application developed for Assignment 4, the final project of the course.

The website combines all previous assignments into one polished and professional application. It presents my background, technical skills, academic projects, GitHub repositories, online courses, and contact information through a responsive and interactive design.

The goal of this project is to demonstrate front-end web development skills using HTML, CSS, and JavaScript while creating a portfolio website ready to share publicly.

## Setup Instructions

### Run Locally

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a browser.

### Recommended Method

Use VS Code Live Server:

1. Open project in Visual Studio Code.
2. Install Live Server extension.
3. Right click `index.html`
4. Select Open with Live Server

## How to Use the Website

### Navigation

Use the navigation bar to move between sections.

### Theme Toggle

Click the Dark Mode / Light Mode button to switch themes.

### Skills Dashboard

View progress bars showing current technical skills.

### Projects Section

- Filter projects by category
- Sort projects alphabetically

### GitHub Section

Enter your details to test validation and success/error messages.
